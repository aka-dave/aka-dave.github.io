Dog API project

View project

In this project I built a dog image viewer that filters the images displayed based on what breed is chosen from a list of breeds. The name of the breeds are extracted from a dog api to form a select element of all the breeds. Once the user selects the breed, a request is made to the API to get a random image of that breed which is then scripted to display directly under the select box. Animations were also created to signal to the user that the image hasn't loaded yet since this is an asynchronous process.

Sum of digits project

View project

In this project I had to create a data structure solution that returns the sum of the biggest pair of integers whose sum are equal. For example, given an array of [25, 52, 41], the output should return 77 since 25 and 52 are 1 pair. If the array has no pairs then it return -1. This project was interestingly challenging to me as I needed to think in a way I have not before. I had to use nested arrays to keep track of the positioning of specific elements. The project was complete in Python programming language.

Password verfication project

View project

For this project I built a solution to data structure problem involving password verfication. The criteria was that for a string S consisting of numbers and/or uppercase and lowercase letters, return the biggest substring that is valid. A valid substring for this case was one that contains atleast one uppercase charactr and no numbers. The solution to this problem became clearer when I thought of it being split into three segments: the initial condition, end condition and everything that happens inbetween.
